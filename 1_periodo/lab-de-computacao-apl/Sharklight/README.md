# Sharklight

Sharlight é um pequeno projeto desenvolvido com um arduino e alguns componentes com o objetivo de acender uma lâmpada por um sensor de movimento.  

Foram usados os seguintes componentes para a construção do projeto:
* Arduino Uno
* Protoboard
* Sensor PIR HC-SR501
* Relé 5V 10A de 1 canal
* Bocal
* Lâmpada

O código do projeto foi feito usando a IDE Arduino IDE e com a biblioteca IRremote, portanto, possa ser que não funcione em IDEs mais convencionais.
